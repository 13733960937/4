文字相关操作

# 设置字体和大小
font = pygame.font.SysFont("SimHei", 25)

# render(text(文本内容), antialias(抗锯齿), color(RGB))，返回文字对象
textobj = font.render("欢迎体验飞机大战游戏", 1, (255, 255, 255) )

# 设置文字矩形对象位置
textrect = textobj.get_rect(centerx = 300, centery = 300)

# 在指定位置绘制指定文字对象
window.blit(textobj, textrect)
  • RGB (R=Red，G=Green，B=Blue）

RGB色彩模式使用RGB模型为图像中每一个像素的RGB分量分配一个0~255范围内的强度值。例如：纯红色R值为255，G值为0，B值为0；灰色的R、G、B三个值相等（除了0和255）；白色的R、G、B都为255；黑色的R、G、B都为0。RGB图像只使用三种颜色，就可以使它们按照不同的比例混合，在屏幕上重现16777216种颜色。

  • 获取字体样式

pygame.font.get_fonts()
